## Description

NestJS test project

## Installation

```bash
$ npm install -g yarn
$ yarn install --frozen-lockfile
```

## Running the app

```bash
# development
$ npm run start # head to localhost:3000

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
